## 🌱 Kezdjük el

Ez a tanfolyam leckéket tartalmaz az AI ügynökök építésének alapjairól. Minden lecke saját témát fed le, így tetszés szerint kezdhetsz bárhol!

A tanfolyam többnyelvű támogatással rendelkezik. Tekintsd meg a [itt elérhető nyelveket](#-multi-language-support).

Ha először építesz Generatív AI modellekkel, nézd meg a [Generatív AI Kezdőknek](https://aka.ms/genai-beginners) tanfolyamunkat, amely 21 leckét tartalmaz GenAI építéséről.

Ne felejtsd el [csillagozni (🌟) ezt a tárolót](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) és [forkolni ezt a tárolót](https://github.com/microsoft/ai-agents-for-beginners/fork) a kód futtatásához.

### Ismerkedj más tanulókkal, kapj választ kérdéseidre

Ha elakadsz vagy kérdésed van az AI ügynökök építésével kapcsolatban, csatlakozz dedikált Discord csatornánkhoz a [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) szerveren.

### Amire szükséged van

A tanfolyam minden leckéje tartalmaz kód példákat, amelyek a code_samples mappában találhatók. Tudsz [forkolni ezt a tárolót](https://github.com/microsoft/ai-agents-for-beginners/fork), hogy saját példányt hozz létre.  

A kód példákban Microsoft Agent Framework és Azure AI Foundry Agent Service V2 használatos:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure fiók szükséges

A tanfolyam a következő Microsoft AI ügynök keretrendszereket és szolgáltatásokat használja:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


További információ a tanfolyam kódjának futtatásáról a [Course Setup](./00-course-setup/README.md) oldalon található.
